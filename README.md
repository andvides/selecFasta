---------------------------------------------------------------
# selectFasta
---------------------------------------------------------------
FASTA or FASTQ select from list

---------------------------------------------------------------
#Running DATMA
---------------------------------------------------------------
selectFasta -list <file with headers to select> -fastq <original fastq file> -fasta_sel
selectFasta -list <file with headers to select> -fasta <original fasta file> -fasta_sel
